## 편집 메모
1. **첫 1초 = 훅 그대로.** 인트로 없이 0초부터 VO + 화면텍스트.
2. **자막 칩**은 컷마다 6~10자, TTS와 동시에 등장.
3. **7번(순차 점등) → 8번(손 CTA)**는 같은 거실 방이라 컷 튐 없이 바로 이어 붙음.
4. **8번 손가락이 가리키는 자리**에 "댓글 '조명' → 링크 DM" 그래픽/스티커 얹기.
5. **BGM**: lo-fi / cozy, 점등 타이밍에 비트 맞추면 좋음.
6. **TTS**: 빠른 숏폼 톤, 1.2~1.3배속 느낌. (ElevenLabs/Typecast 류)
7. **색보정**: 전체 따뜻한 톤 통일 (특히 와이드 시작 컷).
